# Development Audit Playbook

**Status:** Draft complete (two versions)
**Location:** Business Operations/

## What It Is
John's proprietary methodology document that codifies how he conducts a fundraising/development audit for new clients. Two versions exist:

1. **Development-Audit-Playbook.md** - Practitioner guide (~5,200 words, ~15-18 pages formatted). Covers the full methodology concisely. Good for re-reading before engagements.

2. **Development-Audit-Reference-Manual.md** - Comprehensive reference manual (~15,300 words, ~40 pages formatted). Includes step-by-step how-to guidance, 11 reusable templates (T1-T11), benchmark reference tables, glossary, red flags quick reference, and two anonymized case studies.

## Templates Included (Reference Manual)
- T1: Sample Engagement Letter
- T2: Data Request Letter
- T3: Board Fundraising Survey (with scoring guide)
- T4: Donor Data Analysis Worksheet
- T5: Communications Scoring Rubric
- T6: Organizational Health Scoring Rubric
- T7: Fundraising Program Scorecard
- T8: Stewardship Audit Matrix
- T9: Audit Report Outline
- T10: Implementation Roadmap
- T11: Board Presentation Handout
